## Your Behind-the-Scenes Support

Startery uses AI Agents to streamline work across all phases—so humans can do their best work, not busywork.

---

## Types of Agents

<CardGroup cols={2}>
  <Card title="Scoping Agents" icon="magnifying-glass-chart">
    Turn rough ideas into scoped project briefs with timeline, roles, and
    outcomes.
  </Card>
  <Card title="Staffing Agents" icon="people-arrows-left-right">
    Match Operators to projects using skills, availability, and trust signals.
  </Card>
  <Card title="Project Agents" icon="diagram-project">
    Track project flow, flag blockers, generate check-ins, and suggest
    improvements.
  </Card>
  <Card title="Execution Agents (Coming Soon)" icon="wand-magic-sparkles">
    First-draft research, content, design, or code to accelerate delivery and
    let humans refine.
  </Card>
</CardGroup>

<Note>
  Our agents never replace humans. They handle grunt work so Operators can focus
  on what matters.
</Note>
